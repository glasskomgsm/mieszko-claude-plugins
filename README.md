# Mieszko's Claude plugins

Personal plugin marketplace for Claude (Cowork / Claude Code).

## Install on any computer

Settings -> Capabilities -> add marketplace: `<github-username>/mieszko-claude-plugins`, then install **animatronics-workshop**.

## Plugins

- **animatronics-workshop** — mechanism design, Excel parts lists (Allegro / Botland / TME, PLN), free 3D models for Onshape, build plans.
